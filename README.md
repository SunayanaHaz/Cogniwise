# Cogniwise
CogniWise is a zero-hardware cognitive readiness app that helps people understand their brain performance using daily inputs like sleep, stress, hydration, energy, and mental load.

## What it does
- Creates a daily CogniScore
- Explains what is affecting cognitive readiness
- Suggests micro-actions like hydration, breathing, and breaks
- Shows a simple 5-day trend

## Why it matters
Most brain health data already exists, but it is scattered across apps and devices. CogniWise turns that fragmented data into a simple daily readiness signal.

## Built as a Zero-to-One MVP
This version is a front-end prototype designed to demonstrate the product experience quickly and clearly.
